#### Install packages from AUR

```shell
sudo pamac install --no-confirm nfs2se
sudo unzip NFSIISE.zip -d /opt/nfs2se/
sudo sh /opt/nfs2se/convert_to_lowercase /opt/nfs2se/FEDATA /opt/nfs2se/GAMEDATA
```
